# Sidebar Menu Template

A modern, fully responsive sidebar navigation template built with pure HTML, CSS, and a touch of vanilla JavaScript. Perfect for dashboards, admin panels, web apps, or any project that needs a sleek and professional side menu.

![Sidebar Menu Preview](images/preview.png)

## Live Demo

[**Open Live Demo →**](https://fathyhesham.github.io/Sidebar-Menu-Template/)

## Overview

This is a clean and elegant sidebar menu template featuring smooth animations, hover effects, active states, and a mobile-friendly collapsible design. It works perfectly out of the box and can be easily customized to fit any branding or layout.

### Key Highlights:
- Beautiful gradient background & modern glassmorphism effects
- Smooth open/close animation for the sidebar
- Active menu item highlighting
- Responsive design – collapses into a hamburger menu on mobile
- Font Awesome icons for a polished look
- Lightweight & no framework dependencies

## Features

- Fully Responsive (Desktop → Tablet → Mobile)
- Smooth Sidebar Toggle Animation
- Active & Hover States for Menu Items
- Mobile-First Hamburger Menu

## How to Use

### Option 1: Live Demo (Instant)
Just click here → [https://fathyhesham.github.io/Sidebar-Menu-Template/](https://fathyhesham.github.io/Sidebar-Menu-Template/)

### Option 2: Run Locally

**For Developers:**
```bash
git clone https://github.com/FathyHesham/Sidebar-Menu-Template.git
cd Sidebar-Menu-Template
```
Then open `index.html` in your browser.

**For Everyone Else (Zero setup):**
1. Go to the repo → Click **Code** → **Download ZIP**
2. Extract the folder
3. Double-click `index.html` → Opens instantly!

## Customization

- Change colors: Edit the CSS variables in `:root` (in `style.css`)
- Update menu items: Modify the `<ul>` list in `index.html`
- Replace icons: Swap Font Awesome class names
- Adjust width/animation speed: Edit `.sidebar` and `.sidebar.active` styles
